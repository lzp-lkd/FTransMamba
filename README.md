Fig. 1. Illustration of scene understanding in remote sensing images. (a) Comparison between unimodal and multimodal scene understanding. (b) Single encoder fusion: Multimodal data (e.g., RGB+DSM) are fused at the input level or early in the encoder. (c) Dual-branch independent encoding: Separate encoders are used for different modalities, with feature fusion occurring at intermediate layers or in the decoder. (d) Multi-stage cross-modal fusion: Progressive integration of cross-modal features at multiple stages of the network to fully exploit complementary information across scales.
